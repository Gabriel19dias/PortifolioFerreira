Python - Busca Quilometragem entre Ceps

Este projeto em Python visa calcular a distância entre dois CEPs utilizando o Google Maps e obter a imagem de thumbnail do trajeto. A solução utiliza as bibliotecas Selenium para automação do navegador e BeautifulSoup para extração de dados da página do Google Maps. O objetivo é gerar uma planilha Excel contendo informações sobre a distância entre os CEPs, o link direto para o trajeto no Google Maps e a imagem de thumbnail do mapa.
